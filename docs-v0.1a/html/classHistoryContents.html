HistoryContents::create (   $history_id,
  $content_id,
  $source = 'hda' 
)

Create a new HistoryContents component to a given history

Corresponds to the Galaxy API method at: POST /api/histories/{history_id}/contents/{type} Types will be "datasets" by default. The other option "dataset_collection" is currently not supported.

Parameters
history_idThe id of the history to add contents to. To obtain history id's, please refer to the index() function of the history class.
content_idThe id of the content to add to the history. To obtian content_id's please refor to this class's index() function.
Returns
An array containing information about the new History content component.
